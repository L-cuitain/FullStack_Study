## js简答题

 1. 网页的三种弹出框分别是什么？作用分别是什么？
 
    答:
    alert() 提示信息
    confirm()   提示确认
    prompt()    提示输入文本

 2. 用户输入框，获取到值的数据类型是什么？

    答:
    数据类型为String类型


## js编程题
1. 打开网页时弹出：全栈牛逼！！！(只有确定键)

    ```html
    alert("全栈牛逼！！！");
    ```

2. 让用户可以在网页输入年龄

    ```html
    prompt("请输入年龄");
    ```

3. 打开网页弹出：您打开了一个网页，是否继续？（有确定键和取消键）

    ```html
    confirm("您打开了一个网页，是否继续？");
    ```

4. 让用户输入一个数字，实现判断该数字是小数还是整数并告诉用户

    ```js
    var str = prompt("请输入数字");

    var num = parseInt(str);

    if(str == num){
        alert("是整数");
    }else{
        alert("是小数");
    }
    ```
