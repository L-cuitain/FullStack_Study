# package.json
```js
//管理本地安装npm包
//package.json 

//package.json创建
//npm init
//package.json内容
//name   名称
//version 版本

//package.json依赖方式
//dependencies: 在生产环境需要用到的依赖
//devDependencies: 在开发 测试环境需要用到的依赖

//安装参数
//npm install package_name --save  表示将这个包添加到 package.json 的 dependencies
//npm install package_name --save-dev  表示将这个包添加到 package.json 的 devDependencies

//存在package.json文件 运行 npm install 会查找文件中的依赖包
```
