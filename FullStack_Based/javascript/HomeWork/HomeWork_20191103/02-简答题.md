## 二、简答题
## 1.css的三种书写方式
1.行内式:
在标记的style属性中设置css样式
2.嵌入式:
在对页面中各种元素的设置集中写在<head>和</head>之间
2.连接式:
<link href="mystyle.css" rel="stylesheet" type="text/css">

## 2.标题标签有哪些

<h2></h2>
<h3></h3>
<h4></h4>
<h5></h5>
<h6></h6>

## 3.图片标签是哪个？有哪些属性，分别是和什么意思
图片标签为:<img>
属性有:
alt  规定图像的替代文本
src  规定显示图像的url
border  定义图像周围的边框
height  定义图像的高度
width  设置图像的宽度


## 4.段落标签是哪个
段落标签为:<p></p>

## 5.超链接是哪个标签，有什么作用，属性有哪些，分别是什么意思
超链接标签为:
<a></a>  用于从一张页面链接到另一张页面
属性有:
href  规定链接指向的页面的url
download  规定被下载的超链接目标
target  规定在何处打开链接文档
type  规定被链接文档的MIME类型
