3. B/S结构和C/S结构是什么？它们之间有哪些区别和联系？
```js
// B/S架构 (Brower/Server) 是浏览器/服务器交互模式  
// C/S架构 (Cilent/Server) 是客户端/服务器交互模式

//区别
// C/S架构安全性比较好  B/S架构安全性较差
// C/S更新和维护 比较麻烦    B/S更新和维护 比较简单
// C/S有单独的客户端 需要特殊安装  B/S统一客户端 无需特殊安装
```

4. 请求响应的流程?

```js
//1.用户通过浏览器发送请求地址
//2.浏览器对用户输入地址进行分析 得到请求协议类型  协议  端口号  及请求路径
//3.浏览器检查本地缓存 如果本地有则得到该域名对应的ip
//4.浏览器没有会发送DNS系统调用 
//5.DNS系统调用结束 浏览器得到对应ip
//6.浏览器通过一个本地的随机端口  建立与服务器指定端口连接的通道
//7.浏览器将信息发送到服务器中
//8.服务器拿到需要返回给浏览器的数据 将数据返回给浏览器
//9.浏览器拿到服务器发送的数据
//10.浏览器根据数据处理响应的内容
```
